My projects

Developed as a part of a Formal Languages and Compiler Design course.

The aim was to parse and translate a simple imperative language into assembly code. It was very enlightening to gain intuition about how programming languages are created. I was glad to have an opportunity to brush up on assembly programming while writing code for conditional statements, loops, and arithmetical operations.

Check out the project's repository

A Civilization-inspired strategy game with LAN multiplayer developed as a university group project.

I was fully responsible for creating the game's user interface, including an interactive world map with tiles, units, and cities. It was nice to work on my pixel art skills while creating sprites. I also contributed to the game logic, where I got to utilise some algorithmic knowledge. I love these moments in game development where an abstract algorithm comes in handy, so using breadth-first search for determining a unit's move range was very satisfying.

Check out the project's repository

A small group project aimed to produce midi songs using a neural network.

We were all neural network newbies, so the project was a gentle introduction to the subject. I got familiar with the basic concepts around it. My job was to prepare learning sets and turn them into proper input network, which allowed me to learn a bit about midi files.
